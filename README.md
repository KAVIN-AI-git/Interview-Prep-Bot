# 🎤 AI Interview Prep Bot

AI-powered mock interview with real Claude feedback.

## 🚀 Deploy on Streamlit Cloud (Free)

### Step 1 — GitHub
1. GitHub.com → New repository → `interview-prep-bot`
2. Upload both files: `app.py` + `requirements.txt`

### Step 2 — Streamlit Cloud
1. Go to **share.streamlit.io**
2. Connect GitHub → Select repo
3. Main file: `app.py`
4. Click **Deploy**

### Step 3 — Add API Key (IMPORTANT!)
1. In Streamlit Cloud → Your app → **Settings → Secrets**
2. Add this:
```
ANTHROPIC_API_KEY = "sk-ant-your-key-here"
```
3. Get free API key from: **console.anthropic.com**

## ✅ Features
- 🎯 Custom role, level, industry setup
- 🤖 Claude AI generates real interview questions
- ✍️ Type your answer → instant AI feedback
- 📊 Score (1-10) + strengths + improvements
- ✨ Model answer shown after each question
- 🏆 Final results with breakdown
- 🌐 English / Tamil / Hindi support
